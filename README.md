## 🛠️ 開発環境

- Rust: 1.75.0
- Cargo: 1.75.0
- OS: Windows 10

## 🚀 実行方法

```bash
# プロジェクトのビルド
cargo build

# テストの実行
cargo test

# 特定のセクションの実行
cargo run --bin section1
cargo run --bin section2
cargo run --bin section3
```
